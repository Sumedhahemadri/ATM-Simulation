💳 ATM Simulation System (Java)
 

📌 Overview
This project is a simple ATM simulation system built using Java. It demonstrates user authentication, custom exception handling, and basic banking security features in a console-based application.

 
 ⚙️ Features
- User authentication using account number and password
- Custom Exception (`InvalidUserException`) for invalid login attempts
- Maximum 3 login attempts for security
- Automatic account blocking after 3 failed attempts
- Clean and modular Object-Oriented Design

🛠️ Technologies Used
- Java
- OOP Concepts
- Exception Handling
- Scanner Class (User Input)
